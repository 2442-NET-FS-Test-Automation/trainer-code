# Pre-watch — Algorithms Primer

**Watch before Wednesday's demo.** Tomorrow the thread steps away from the API for a day of algorithms and
data structures — the vocabulary below is assumed from the first minute, and the demo moves fast through
implementations. Come with questions.

- [ ] **Big-O notation** — what "grows like n" vs "grows like log n" means and why constants are ignored.
- [ ] **Common Algorithms: Searching and Sorting** — linear vs binary search, and the simple sorts
      (bubble, insertion, selection) by shape.

*Time: roughly 25-35 minutes total. You do not need to memorize implementations — build the mental map, so
that when the live code halves a search range or bubbles a value to the end, you recognize the move. The
searching/sorting ideas return with real stakes on Friday-week, when a sorted report turns ranking into a
binary search.*
